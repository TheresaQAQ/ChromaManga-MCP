推理参数

滑动调节 · 应用后 Agent 会自动回退并重跑
guidance_scale 6.0

提示词遵循度。颜色太淡 → 调高（7-8）；过饱和 → 调低（4-5）

num_inference_steps 20

推理步数。细节不够 → 调高（25-30）；推理慢 → 调低（15-18）

controlnet_scale 1.1

线稿控制强度。线稿不清 → 调高（1.2-1.4）；太死板 → 调低（0.8-0.95）
